# 📊 Dashboard de Vendas com Python

Este projeto consiste em um dashboard interativo desenvolvido para análise de dados de vendas, com foco em transformar dados em informações úteis para tomada de decisão.

## 🚀 Tecnologias utilizadas
- Python
- Streamlit
- Pandas

## 📈 Funcionalidades
- Cálculo de faturamento total
- Ticket médio
- Análise de vendas por loja
- Análise de vendas por produto
- Visualização de dados em gráficos

## 🎯 Objetivo
Aplicar na prática conceitos de análise de dados e desenvolvimento de dashboards, reforçando a capacidade de manipulação, interpretação e apresentação de dados.
